- **Attribution:** Any modification must clearly state: "Modification based on original work by [@ALRI GROUP](https://github.com/alrigroup/about)".

- **Restriction:** Selling or monetizing the work is strictly prohibited without express authorization.

### 2. ARGLFU (Free to Use)
*Designed for public utilities and standalone tools.*

- **Integrity:** Modifying or adapting the code is strictly prohibited without prior authorization from the group.
- **Safety:** Ensures end-users receive the official and secure version verified by [ALRI Group](https://github.com/alrigroup/about).

### 3. ARGLR (Reserved)

- **Warranty:** Technical support and bug fixes are guaranteed as long as the code remains **unmodified**.
